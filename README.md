# Basic Project #

Ejemplo usando Django y Backbone.js

### Requirements ###

* Virtualenv or virtualenvwrapper
* Django 1.6

### Create a virtualenv ###

* Virtualenv

    $ virtualenv env
    $ source env/bin/activate

* Virtualenvwrapper

    $ mkvirtualenv env
    $ workon env
    $ deactivate

### How to run the project ###

    $ python manage.py runserver
