# Basic Project #

Ejemplo usuario Django y Backbone.js

### Requirements ###

* Virtualenv or virtualenvwrapper
* Django 1.6

### Create a virtualenv ###

* Usando virtualenv

    $ virtualenv env
    $ source env/bin/activate

* Usando virtualenvwrapper

    $ mkvirtualenv env
    $ workon env

### How to run the project ###

    $ python manage.py runserver --settings=myproject.settings.local
